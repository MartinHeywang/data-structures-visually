Hash Map

A hash map can also be referred as an 'object' or a 'map' in some programming languages.
It is a group of key-value pairs. It is much like an array that uses text to index its values rather the integers.
The hashing function is the function that transforms the key (text) to a valid integer. A hash map uses an array under the hood.
