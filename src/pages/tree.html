Tree

A tree is a group of nodes in which each node can have multiple children nodes.
Here, we are facing a binary tree, because each node can only have 0 to 2 nodes, but you could imagine trees with much more children than that.
A tree is often useful when you want to create some kind of hierarchy, but you can also find it where you need to represent a recursive model (data that repeats itself inside of it), e.g parentheses inside a mathematical expression.
